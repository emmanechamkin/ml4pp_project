# Read me 

Final project directory structure:

- `EC2_tagging` contains all work to have CoreNLP automatically classify tweets
- `mturk_coding` contains all work to evaluate mturk data 
- `proposal` contains work I did for my proposal
- `readings` includes pdfs of the readings I found most novel 
- `report` includes my report, final presentation, and latex work
- `tweet_extraction` includes code and data from interfacing with the twitter API
- `utils` includes twitter credentials

Please see the readme files in each folder for more information.
